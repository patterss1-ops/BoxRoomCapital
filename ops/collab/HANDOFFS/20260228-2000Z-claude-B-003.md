## What was built

### New modules

1. **`fund/nav.py`** — Daily NAV calculation
   - `calculate_fund_nav()` — aggregates positions + cash across all active broker accounts
   - `calculate_sleeve_navs()` — per-sleeve NAV breakdown with proportional cash allocation
   - `persist_nav_report()` — writes to fund_daily_report + sleeve_daily_report tables
   - `run_daily_nav()` — end-of-day job returning summary dict for Telegram
   - Daily return % from previous fund_daily_report
   - High water mark tracking and drawdown calculation

2. **`fund/report.py`** — Fund performance reporting
   - `generate_daily_report()` — single-day fund report with sleeves and risk
   - `generate_period_report()` — rolling period (7d/30d/90d) with stats
   - `format_report_text()` — deterministic text output for Telegram/console
   - Annualised volatility, max drawdown, positive/negative day counts

3. **`risk/portfolio_risk.py`** — Portfolio-level risk metrics
   - `calculate_portfolio_risk()` — heat, concentration, leverage, margin, VaR
   - `generate_risk_verdict()` — deterministic GREEN/AMBER/RED with alerts
   - `get_risk_briefing()` — B-004 integration contract (all 10 fields)
   - `get_position_risk_details()` — per-position drill-down
   - `run_daily_risk()` — end-of-day risk job

### Schema additions (additive only)

All three tables use `CREATE TABLE IF NOT EXISTS` — zero migration risk.

| Table | Unique Key | Purpose |
|-------|-----------|---------|
| `fund_daily_report` | `report_date` | Daily fund NAV, return, drawdown, HWM |
| `sleeve_daily_report` | `report_date, sleeve` | Daily per-sleeve breakdown |
| `risk_daily_snapshot` | `snapshot_date` | Daily heat, margin, concentration, leverage, VaR |

### B-004 integration contract

`get_risk_briefing()` returns:
```python
{
    "fund_nav": float,
    "day_pnl": float,
    "drawdown_pct": float,
    "gross_exposure_pct": float,
    "net_exposure_pct": float,
    "cash_buffer_pct": float,
    "open_risk_pct": float,
    "generated_at": str,  # ISO UTC
    "status": str,        # GREEN/AMBER/RED
    "alerts": [{"severity", "code", "message", "action"}],
    "limits": [{"rule", "threshold", "warning", "current"}],
}
```
