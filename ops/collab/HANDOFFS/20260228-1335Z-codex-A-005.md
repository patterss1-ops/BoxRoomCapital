- Regression impact: moderate-low. Changes are additive, but they introduce new schema objects and aggregation/reconciliation logic that can affect operator interpretation if broker ingestion payloads are malformed.
- Rollback note: revert commit `088fa3e` to remove A-005 runtime/schema additions.
