❯ what is the API Secret is that the same as the private key?                                                                                                                                                                              
                                                                                      
● Yes — for Kraken, the API Secret is the Private Key. When you create an API key pair on Kraken (under Settings > API), they give you two values:                                                                                         
                                                                                                                                                                                                                                           
  - API Key — the public identifier → KRAKEN_API_KEY                                                                                                                                                                                       
  - Private Key (also called API Secret) — the signing secret → KRAKEN_API_SECRET                                                                                                                                                          
                                                                                                                                                                                                                                           
  Same thing, different names.
